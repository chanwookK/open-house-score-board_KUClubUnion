# 📖 프로젝트 소개

### 📄 프로젝트 개요 및 목적

이 프로젝트는 **2024 오픈하우스** 기획의 일환으로 진행되는 웹 개발 프로젝트로, 다양한 동아리들의 점수를 실시간으로 공개하고 계산하는 사이트입니다. 본 웹 사이트는 행사 중 실시간 점수 업데이트 및 공개를 통해 참가자의 경험을 향상시키는 것을 목표로 합니다.

### ✨ 주요 기능

- 기능 1: 실시간 스코어 보드 업데이트
- 기능 2: 실시간 스코어 업데이트
- 기능 3: 실시간 메시지 전달
